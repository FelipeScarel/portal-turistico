Brasil Encantado

O Brasil Encantado é um site desenvolvido com o objetivo de apresentar destinos turísticos brasileiros, destacando experiências únicas, cultura, natureza e gastronomia.

Sobre o Projeto

O site foi criado como um projeto acadêmico de Engenharia de Computação, com foco em desenvolvimento web utilizando HTML e CSS. A proposta é simular uma agência de viagens especializada em turismo de experiência no Brasil.

Funcionalidades
Página inicial
Catálogo de destinos turísticos pelo Brasil
Seção de gastronomia com conteúdos culturais e vídeos
Página de contato com formulário para atendimento
